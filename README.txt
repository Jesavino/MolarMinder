An Android Application for tracking when a user wearing a 
smart device is brushing their teeth. Handles the real time
data smoothing using a sliding window. Currently is configured 
to connect to a specific Shimmer wrist mounted unit, but this
functionality can easily be extended to connect to any device 
which provides a stream of accelerometer data. 

Connects a single Shimmer unit calibrated accelerometer data can be seen on the LogCat.
